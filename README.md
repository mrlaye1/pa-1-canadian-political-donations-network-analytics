# Canadian Political Donations Network Analytics

## Complex Network Analysis of Canadian Political Donations (2003–2018)

Advanced graph-based analysis of Canadian political donations using NetworkX, graph theory, centrality metrics, funding concentration analytics, community detection, Jaccard similarity analysis, interactive network visualizations, and an executive decision-support dashboard.

---

# Overview

This project applies advanced **Complex Network Analysis** techniques to study Canadian political donation patterns between 2003 and 2018.

Rather than viewing donations as isolated transactions, the project models relationships between:

- Political Parties
- Forward Sortation Areas (FSAs)
- Provinces
- Donor Communities

The project builds multiple network representations and analyzes:

- Political influence
- Fundraising reach
- Geographic donor concentration
- Community structures
- Similarity between donor regions
- Similarity between political parties
- Fundraising concentration risk

The final deliverable is a fully interactive Executive Dashboard designed for decision-makers, researchers, political analysts, and policy professionals.

---

#  Problem Definition

Political fundraising success depends on understanding:

- Which parties dominate fundraising networks
- Which regions generate the most donations
- Whether funding sources are diversified or concentrated
- Which donor communities drive fundraising activity
- How political parties compete for similar donor regions
- Which geographic areas present future growth opportunities

Traditional donation reporting focuses primarily on totals and rankings.

This project extends beyond descriptive reporting by applying network science techniques to uncover hidden structural relationships within Canada's political fundraising ecosystem.

---

# Objectives

The project was designed to:

1. Build political donation networks from Canadian donation records
2. Measure political influence using graph centrality metrics
3. Identify high-value donor regions
4. Evaluate fundraising concentration risk
5. Detect donor communities and fundraising clusters
6. Measure similarity between political parties
7. Measure similarity between donor regions
8. Generate advanced interactive visualizations
9. Produce a stakeholder-focused executive dashboard

Measures overall network influence.

Measures direct connectivity.

Measures fundraising strength.

Measures bridge importance.

Measures network accessibility.

Measures influence quality.

# Limitations

- Results depend on available donation records.
- Community structures may evolve as additional years are included.
- Similarity metrics measure network overlap and should not be interpreted as political alignment.
- Centrality metrics measure structural influence rather than electoral success.
